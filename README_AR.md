# ESC Smart Wayfinder — نموذج أولي جاهز للنشر

## ما الذي يحتويه؟
- موقع عربي/إنجليزي يعمل على الهاتف دون تسجيل دخول.
- إرشاد خطوة بخطوة: صعود السلم، يمين، يسار، متابعة السير، الوصول.
- قراءة صوتية للتعليمات.
- خريطة مسار مختصرة واختيار موقع حالي عند الضياع.
- يعمل كتطبيق PWA ويمكن إضافته إلى الشاشة الرئيسية.
- يعمل دون إنترنت بعد فتحه أول مرة.
- يدعم نقاط بداية مختلفة من خلال الرابط، مثل:
  - `?start=entrance`
  - `?start=first-floor`
  - `?start=corner`
  - `?start=corridor`
  - `?start=destination`

## تجربة النموذج على الكمبيوتر
يجب فتحه عبر خادم محلي، لا بالنقر على الملف مباشرة:

```bash
cd ESC_Smart_Wayfinder
python -m http.server 8000
```

ثم افتح: `http://localhost:8000`

## النشر
يمكن رفع المجلد كما هو إلى خادم الجامعة أو GitHub Pages أو Netlify. يفضل رابط رسمي قصير مثل:
`https://edu.alexu.edu.eg/esc-wayfinder/`

## تعديل البيانات
افتح ملف `config.js` وعدّل:
- نصوص الاتجاهات.
- رقم اتصال المركز.
- عدد نقاط المسار أو ترتيبها.

## إنشاء رموز QR بعد الحصول على الرابط النهائي
```bash
pip install qrcode[pil]
python generate_qr.py https://YOUR-FINAL-DOMAIN/esc-wayfinder/
```
ستُنشأ خمسة رموز في مجلد `qr-codes`، وكل رمز يفتح التطبيق من نقطة مختلفة.

## قبل الإطلاق الفعلي
يلزم تزويد التطبيق بالآتي:
1. صور حقيقية لنقاط الطريق من زاوية نظر المتدرب.
2. تأكيد الأدوار والاتجاهات النهائية.
3. رقم هاتف أو واتساب رسمي للمساعدة.
4. الرابط الرسمي الذي سيستضيف التطبيق.

- تم تحديث رقم الاتصال بالمركز إلى: `+201124239879` (محليًا: `01124239879`).
